Summarizer API Documentation

Summarizer provides an API to programmatically generate video summaries. You can use this API by making GET requests with specific parameters.

API Endpoint

GET /summarize/

Parameters

  • id: YouTube video ID (required)
  • percent: Percentage of text to summarize (required)
  • choice: Summarization method choice (required)

Example Usage

Example URL: /summarize/?id=VIDEO_ID&percent=50&choice=spacy-sum
